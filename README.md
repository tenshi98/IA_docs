# Prompts para uso general

## Normales

- Consultar por un tema en especifico

```bash
ROL: Eres un profesor de bachillerato de matemáticas
TAREA: Explícame que el teorema de pitágoras
CONTEXTO: No soy muy bueno en matemáticas, por lo que me gustaría sencillez
RAZONAMIENTO: Quiero que lo hagas paso a paso, con ejemplos visuales y terminar con una conclusión
FORMATO DE SALIDA: Dame dos párrafos explicándolo y la formula matemática
LIMITE: Cuando me des tres ejemplos
```

## Creación de proyectos

- Creación de un sistema de autenticación completo

```bash
Crea un sistema de autenticación completo con:
- Express backend con endpoints /register, /login, /profile
- JWT para tokens
- Hash de passwords con bcrypt
- Middleware de autenticación
- Validación de datos
- Manejo de errores apropiado
- Test con Jest
```
